Untuk menjalankan Objek Bandara 3D menggunakan Dev c++, mka ada settingan tambahan agar suara bisa terdengar
dan aplikasi bisa berjalan.

1.Copy seluruh file .h yg ada di folder Include ke  Dev c++->Include->Paste.
2.Copy seluruh file yg ada di folde lib ke Dev c++ -> lib -> Paste.
3.Copy file fmod.dll ke system32 anda.
4.Pada source code Objek 3D bandara, tambahkan parameter -lfmod.
